Overview

This project is a simple Electric Vehicle (EV) Chatbot built using Python. The chatbot is designed to answer basic EV-related queries such as battery life, charging time, range, cost, and efficiency. It uses rule-based logic to simulate conversations and provide quick responses to user inputs.

Objective

To develop a chatbot that can interact with users and provide information related to electric vehicles, enhancing automation and user engagement through natural language communication.
